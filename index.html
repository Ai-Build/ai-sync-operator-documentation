IO

Load Mesh

Loads a file from the files library of the project that represents a mesh geometry. Supported file format: obj.

Inputs:

  • In (Geometry file)

Outputs:

  • Out (Mesh)

Load Polylines

Loads a file from the files library of the project that represents polyline geometry . Supported file format: aib.

Inputs:

  • In (Geometry file)

Outputs:

  • Out (Polylines)

SLICERS

Angular Slicer

Operator for slicing the geometry at the specified angle in relation to the horizontal plane and adding multiple walls and infill.

Inputs:

  • In (Mesh)
  • LayerHeight (decimal) | Spacing between planes for slicing the geometry
  • PlaneRotX (decimal) | Plane angle in relation to the horizontal plane in workspace X-axis defined in degrees
  • PlaneRotY (decimal) | Plane angle in relation to the horizontal plane in workspace Y-axis defined in degrees
  • PlaneRotZ (decimal) | Plane angle in relation to the horizontal plane in workspace Z-axis defined in degrees
  • WallCount (integer) | Number of walls to be created
  • OuterWidth (decimal) | Thickness of outer walls defined in mm
  • InnerWidth (decimal) | Thickness of inner walls defined in mm
  • CompensateWidth (boolean) | If enabled, the outer wall will be created by half the amount of the outer width inwards to compensate for the extruded material's additional thickness
  • EnableInfill (boolean) | If enabled, infill structure will be generated
  • InfillDensity (decimal) | Infill density set in values between 0-1 for 0-100%
  • InfillWidth (decimal) | Infill line thickness defined in mm
  • InfillAngle (decimal) Infill rotation angle in degrees
  • AlternateRotation (boolean) | If enabled, infill is rotated by 90 degrees in every alternate layer

Outputs:

  • Out (Polylines)

Horizontal Slicer

Operator for slicing the geometry horizontally and adding multiple walls and infill.

Inputs:

  • In (Mesh)
  • LayerHeight (decimal) | Spacing between planes for slicing the geometry defined in mm
  • WallCount (integer) | Number of walls to be created
  • OuterWidth (decimal) | Thickness of outer walls defined in mm
  • InnerWidth (decimal) | Thickness of inner walls defined in mm
  • CompensateWidth (boolean) | If enabled, the outer wall will be created by half the amount of the outer width inwards to compensate for the extruded material's additional thickness
  • EnableInfill (boolean) | If enabled, infill structure will be generated
  • InfillDensity (decimal) | Infill density set in values between 0-1 for 0-100%
  • InfillWidth (decimal) | Infill line thickness defined in mm
  • InfillAngle (decimal) Infill rotation angle in degrees
  • AlternateRotation (boolean) | If enabled, infill is rotated by 90 degrees in every alternate layer

Outputs:

  • Out (Polylines)

Multi Planar Slicer

Slices the input solid geometry with multiple planes and returns the intersection as polylines. Takes as input two planes (see Create Plane operator) and the number of total planes which are calculated as interpolation between the two input planes.

Inputs:

  • In (Mesh)
  • StartPlane (Plane)
  • EndPlane (Plane)
  • PlaneCount (int) | Total number of planes to be generated
  • WallCount (integer) | Number of walls to be created
  • OuterWidth (decimal) | Thickness of outer walls defined in mm
  • InnerWidth (decimal) | Thickness of inner walls defined in mm
  • CompensateWidth (boolean) | If enabled, the outer wall will be created by half the amount of the outer width inwards to compensate for the extruded material's additional thickness
  • EnableInfill (boolean) | If enabled, infill structure will be generated
  • InfillDensity (decimal) | Infill density set in values between 0-1 for 0-100%
  • InfillWidth (decimal) | Infill line thickness defined in mm
  • InfillAngle (decimal) Infill rotation angle in degrees
  • AlternateRotation (boolean) | If enabled, infill is rotated by 90 degrees in every alternate layer

Outputs:

  • Out (Polylines)

Non Planar Slicer

Slices the input solid geometry in a non planar manner. Two reference meshes are required as boundary slicing guides, with all the in-between slicing guides calculated as interpolation between the two input geometries.

Inputs:

  • In (Mesh)
  • GuideA (Mesh) | First mesh to be used as a slicing guide to intersect with the input geometry
  • GuideB (Mesh) | Second mesh to be used as a slicing guide
  • Count (integer) | Total number of meshes to be generated for slicing with the base geometry
  • WallThickness (decimal) | Thickness of the sliced geometry defined in mm

TOOLPATH

Toolpath Generator Filament

Generates a toolpath from the input polylines for a filament extruder.

Inputs:

  • Design (Polylines) | Input polylines to be used for toolpath generation
  • Enable Axis Limits (boolean) | If enabled, the computation will check if the robot exceeds its axis limits. If it exceeds them, the toolpath generation will fail.
  • Enable Workspace Limits (boolean) | If enabled, the computation will check if the robot excedds its workspace limits. If it exceeds them, the toolpath generation will fail.
  • Material (selection) | The type of material to be used
  • Nozzle T (integer) | Nozzle T defined in Celsius degrees
  • First Layer Nozzle T (integer) | Nozzle T for the first layer defined in Celsius degrees
  • Bed T (integer) | Heated bed temperature defined in Celsius degrees
  • First Layer Bed T (integer) | Heated bed temperature for the first layer defined in Celsius degrees
  • Cell T (integer) | Heated enclosure temperature defined in Celsius degrees
  • First Layer Cell T (integer) | Heated enclosure temperature for the first layer defined in Celsius degrees
  • Max Extrusion Rate (integer) | Maximum extrusion rate of the extruder defined in steps/s
  • First Layer Max Extrusion Rate (integer) | Maximum extrusion rate for the first layer defined in steps/s
  • Outer Wall Speed (decimal) | Robot speed for the outer wall linetype polylines defined in mm/s
  • Inner Wall Speed (decimal) | Robot speed for the inner wall linetype polylines defined in mm/s
  • Infill Speed (decimal) | Robot speed for the infill linetype polylines defined in mm/s
  • Skin Speed (decimal) | Robot speed for the skin linetype polylines defined in mm/s
  • Support Speed (decimal) | Robot speed for the support linetype polylines defined in mm/s
  • Scan Distance (decimal) | If scan distance is bigger than zero, then before breaking from a closed polyline, the robot will continue moving along the polyline path for the speficied scan distance and with the specified scan speed so that the nozzle gets clean from any residual material. Defined in mm.
  • Scan Speed (decimal) | Robot speed while scanning the geometry defined in mm/s.
  • Jump Speed (decimal) | The speed that the robot will jump when breaking from a polyline in order to travel to the next polyline defined in mm/s.
  • Wait Before Extrusion (decimal) | The delay in seconds between switching on extrusion and moving along the path of a polyline to account for the delay between sending the instruction to the extruder to switch the extrusion on and the material to come out of the nozzle.
  • Wait After Open Polyline (decimal) | The delay in seconds between switching off extrusion and breaking from the open polyline at the end of its path.
  • Vertical Offset Before Start (decimal) | The vertical distance in mm from the first point of a new polyline after a travel move.
  • Jump Distance (decimal) | The vertical jump distance in mm at the end of a polyline before a travel move.
  • Line Continuity Limit (decimal) | The maximum distance between the last point of a polyline and the first point of the next polyline in order to be considered continuous path without adding a travel move between them. Extrusion will remain on if the two points are within the continuity limit.
  • Cooler Speed (selection) | Defines the cooling speed
  • Enable Minimum Layer Time (boolean) | If enabled, the robot and extrusion speed will be adjusted to ensure that each layer is printed no sooner than the minimum specified layer time.
  • Minimum Layer Time (decimal) | Minimum layer time defined in seconds.

Outputs:

  • Toolpath (file) | The generated toolpath that can be streamed or downloaded.
  • Duration (time) | The estimated printing duration of the toolpath defined in hours, minutes and seconds.
  • Weight (decimal) | The estimated weight of the printed object based on the generated toolpath, defined in kg.

DATA OPERATIONS

Combine Data

Combines two polyline outputs into one. DataA is added first.

Example:

Data A: [pl1, pl2, pl3], Data B: [pl4, pl5, pl6]

Result: [pl1, pl2, pl3, pl4, pl5, pl6]

Inputs:

  • DataA (Polylines)
  • DataB (Polylines)

Outputs:

  • Out (Polylines)

Reverse Data

Reverses the order of the elements in the input data.

Example:

Input data: [pl1, pl2, pl3, pl4, pl5, pl6]

Result: [pl6, pl5, pl4, pl3, pl2, pl1]

Inputs:

  • In (Polylines)

Outputs:

  • Out (Polylines)

Set Layer

Sets the layer information for the input polylines.

Inputs:

  • In (Polylines)
  • Layer (integer)

Outputs:

  • Out (Polylines)

Set Layer Height

Sets the layer height information for all the points of the input polylines in mm.

Inputs:

  • In (Polylines)
  • LayerHeight (decimal)

Outputs:

  • Out (Polylines)

Set Line Thickness

Sets the line thickness information for all the points of the input polylines in mm.

Inputs:

  • In (Polylines)
  • LineThickness (decimal)

Outputs:

  • Out (Polylines)

Set Line Type

Sets the line type information for the input polylines.

Inputs:

  • In (Polylines)
  • LineType (selection)

Outputs:

  • Out (Polylines)

Set Orientation

Sets the tool a, b,c rotation values from the home position for all the points of the input polylines in degrees.

Inputs:

  • In (Polylines)
  • LineThickness (decimal)

Outputs:

  • Out (Polylines)

Set Speed Multiplier

Sets the speed multiplier for all the points of the input polylines. Default value is 1. A smaller speed multiplier will result in slower extrusion and robot speed, while a higher value will result in faster extrusion and robot speed.

Inputs:

  • In (Polylines)
  • SpeedMultiplier (decimal)

Outputs:

  • Out (Polylines)

WeaveData

Weaves the data of two lists of polylines with the same number of elements.

Example:

Input lists: [pl1, pl2, pl3], [pl4, pl5, pl6]

Result: [pl1, pl4, pl2, pl5, pl3, pl6]

Inputs:

  • DataA (Polylines)
  • DataB (Polylines)

Outputs:

  • Out (Polylines)

FILTERS

Filter By Analysis Data

Filters the input data based on the range of the specified analysis data.

Inputs:

  • In (Polylines)
  • Data (decimal list) | Analysis data of the input polylines
  • MinValue (decimal) | Lower limit to filter from the analysis data
  • MaxValue (decimal) | Upper limit to filter from the analysis data

Filter By Average Height

Filters the input polylines based on the specified range of polyline average height. A maxHeight value of -1 returns the maximum upper limit available.

Inputs:

  • In (Polylines)
  • MinHeight (decimal)
  • MaxHeight (decimal)

Outputs:

  • Out (Polylines) | The polylines within the specified range
  • OutInverse (Polylines) | The polylines outside the specified range

Filter By Index Range

Filters the input polylines based on the specified range of indices. An endIndex value of -1 returns the biggest index available.

Inputs:

  • In (Polylines)
  • StartIndex (integer)
  • EndIndex (integer)

Outputs:

  • Out (Polylines) | The polylines within the specified range
  • OutInverse (Polylines) | The polylines outside the specified range

Filter By Layer

Filters the input polylines based on the specified range of layers. Layer data needs to be present in the input polylines in order for the operator to work. An endLayer value of -1 will returns the biggest layer available.

Inputs:

  • In (Polylines)
  • StartLayer (integer)
  • EndLayer (integer)

Outputs:

  • Out (Polylines) | The polylines within the specified range
  • OutInverse (Polylines) | The polylines outside the specified range

Filter By Layer Height

Filters the input polylines based on the specified range of layer height. Operator will only work if layer height information is stored in the input data.

Inputs:

  • In (Polylines)
  • MinValue (decimal)
  • MaxValue (decimal)

Outputs:

  • Out (Polylines)

Filter By Length

Filters the input polylines based on the specified range of polyline length.

Inputs:

  • In (Polylines)
  • MinLength (decimal)
  • MaxLength (decimal)

Outputs:

  • Out (Polylines) | The polylines within the specified range
  • OutInverse (Polylines) | The polylines outside the specified range

Filter By Linetype

Separates the input polylines into different streams based on their linetype information.

Inputs:

  • In (Polylines)

Outputs:

  • OuterWall (Polylines)
  • InnerWall (Polylines)
  • Infill (Polylines)
  • Support (Polylines)
  • Skin (Polylines)

Filter Closed Polylines

Separates the input polylines into two streams based on the open/closed polyline condition.

Inputs:

  • In (Polylines)

Outputs:

  • OutClosed (Polylines)
  • OutOpen (Polylines)

Filter Internal Polylines

Separates the input polylines into two streams based on whether they are internal/external on the layer.

Inputs:

  • In (Polylines)

Outputs:

  • OutInternal (Polylines)
  • OutExternal (Polylines)

Filter Difference

Takes a list of polylines and a sublist and removes the sublist's polylines from the list.

Example: List [ a, b, c, d, e ], Sublist [ b, d ] Out [ a, c, e ]

Inputs:

  • List (polylines) | Base list of polylines
  • Sublist (polylines) | List of polylines to be subtracted from the base list

Outputs:

  • Out (polylines)

GEOMETRY

Align Seams

Aligns the input polyline seams based on proximity to a reference point. Works only for closed polylines. Open polylines remain unmodified.

Inputs:

  • In (Polylines)
  • RefPtX (decimal) | Reference point coordinate in workspace X-axis in mm
  • RefPtY (decimal) | Reference point coordinate in workspace Y-axis in mm
  • RefPtZ (decimal) | Reference point coordinate in workspace Z-axis in mm
  • ClosestNeighbour (boolean) | If closest neighbour is disabled, the closest point of a polyline to the reference point becomes the new seam. If enabled, the first polyline's new seam is the closest point to the reference point and each consecutive element's new seam is the closest point to the previous element's new seam.
  • IgnoreZ (boolean) | If enabled, the reference point Z value is ignored and the reference point has coordinates (refPtX, refPtY, polylineAverageHeight)

Outputs:

  • Out (Polylines) | Polylines with aligned seams

Brim

Creates a brim for the input solid geometry on the horizontal plane located at workspace z zero. If the input geometry is not on z=0, no brim will be created. Returns a list of a contours as polylines.

Inputs:

  • In (Mesh)
  • LineWidth (decimal) | Thickness of brim lines defined in mm
  • LineCount (integer) | Number of brim lines

Outputs:

  • Out (Polylines)

Close Polylines

Returns a list of closed polylines. If any open polylines are detected, it makes them closed by connecting their endpoints with a line segment.

Inputs:

  • In (Polylines)

Outputs:

  • Out (Polylines)

Connect Open Polylines

Detects open polylines and connects them together, either per list or per layer. There is the option to make the output polylines closed.

Inputs:

  • In (Polylines)
  • ConnectPerLayer (boolean) | If enabled, open polylines will be grouped in layers and the connection operation will be applied per layer. If disabled, all the open polylines will connect into one polyline.
  • MakeClosed (boolean) | If enabled, the resulting connected polylines will become closed by adding a line segment that connects the first with the last point of each polyline.

Outputs:

  • Out (Polylines)

Contours At Plane

Returns the intersection between a plane and a solid geometry as polylines.

Inputs:

  • In (Mesh)
  • GuidePtX (decimal) | X coordinate of plane origin in mm
  • GuidePtY (decimal) | Y coordinate of plane origin in mm
  • GuidePtZ (decimal) | Z coordinate of plane origin in mm
  • PlaneRotX (decimal) | Plane angle in relation to the horizontal plane in workspace X-axis around the guide point in degrees
  • PlaneRotY (decimal) | Plane angle in relation to the horizontal plane in workspace Y-axis around the guide point in degrees
  • PlaneRotZ (decimal) | Plane angle in relation to the horizontal plane in workspace Z-axis around the guide point in degrees

Outputs:

  • Out (Polylines)

Create Plane

Creates a plane by defining the plane origin and its rotations around the world x,y,z axis from that point.

Inputs:

  • OriginPtX (decimal) | Origin point x coordinate
  • OriginPtY (decimal) | Origin point y coordinate
  • OriginPtZ (decimal) | Origin point z coordinate
  • RotationX (decimal) | Rotation around the world-x axis defined in degrees
  • RotationY (decimal) | Rotation around the world-y axis defined in degrees
  • RotationZ (decimal) | Rotation around the world-z axis defined in degrees
  • DisplaySize (decimal) | Plane edge length for visualization purposes defined in mm

Flatten Mesh Base

Snaps the vertices of the mesh with the minimum z within the specified tolerance in order to flatten the bottom part of the mesh.

Inputs:

  • In (Mesh)
  • ToleranceZ (decimal)

Outputs:

  • Out (Polylines)

Get Naked Edges From Mesh

Returns the naked edges (borders) of an open mesh as a list of polylines. If the input mesh is closed, the output of this operator will be empty.

Inputs:

  • In (Mesh)

Outputs:

  • Out (Polylines)

Infill 2D

Creates a 2D infill inside the input list of polylines. If a polyline in the list is open it will be excluded from the calculations. The infill pattern is a continuous zig zag. If layer data is available, the infill is applied per layer instead of per polyline. Each alternate layer is rotated by 90 degrees.

Inputs:

  • In (Polylines)
  • Density (decimal) | Infill density. Range is 0-1 (0% - 100%)
  • LineWidth (decimal) | Thickness of infill lines defined in mm
  • Rotation (decimal) | Pattern rotation in relation the workspace XY-plane in degrees
  • PerimeterGap (decimal) | Offset distance from the input polylines defined in mm
  • AlternateRotation (boolean) | If enabled, infill is rotated by 90 degrees in every alternate layer

Outputs:

  • Out (Polylines)

Join Meshes

Returns the combined data of two solid geometries. No union operation is performed and the geometries remain unmodified.

Inputs:

  • InputA (Mesh)
  • InputB (Mesh)

Outputs:

  • Out (Polylines)

Make Uniform

Unifies the direction of the closed input polylines.

Inputs:

  • In (Polylines)

Outputs:

  • Out (Polylines)

Offset

Offsets the input polylines by a specified distance. Positive distance offsets internally and negative distance externally. Operator works only for closed polylines.

Inputs:

  • In (Polylines)
  • Distance (decimal) | Offset distance in mm
  • Count (integer) | Number of offset operations
  • InputType (selection) | If POLYLINE is selected, the offset operation will be applied per polyline. If LAYER is selected, the offset operation will be applied per layer, so the union of the polylines on a layer will be returned.

Outputs:

  • Out (Polylines)

Planes from Polylines

Creates planes from a list of input planar polylines.

Inputs:

  • In (Polylines)

Outputs:

  • Out (Planes)

Primer

Adds a primer in the input polyline geometry. A primer is a line of specified length that is located on the base layer at a distance from the geometry and it is always printed first in order to ensure that the extruder nozzle is clean from any dry material before beginning printing the actual object geometry.

Inputs:

  • In (Polylines) | Input polylines
  • PrimerLength (decimal) | The length of the primer line in mm
  • DistanceFromGeometry (decimal) | The distance of the primer from the geometry in mm
  • PrimerThickness (decimal) | Thickness of the primer line in mm
  • PrimerHeight (decimal) | Height of the primer line in mm

Outputs:

  • OutMerged (Polylines) | The primer geometry merged with the input geometry
  • Out (Polylines) | The pimer geometry

Random Seams

Randomizes the position of input polylines' seams.

Inputs:

  • In (Polylines)

Outputs:

  • Out (Polylines)

Rebuild Polylines

Rebuilds the input polylines by increasing the point density at the specified density.

Inputs:

  • In (Polylines) | Input polylines
  • MaxSpacing (decimal) | Maximum spacing between points. If any two points of a polyline are spaced apart more than the defined maxSpacing, a new point will be added between them

Outputs:

  • Out (Polylines)

Reduce Density

Reduces the point density of the input polylines with the option of preserving sharp features of the geometry.

Inputs:

  • In (Polylines) | Input polylines
  • MinPtSpacing (decimal) | The minimum spacing between polyline points. If the distance between two neighbour points is smaller than the minPtSpacing, the second point will be removed
  • MaxAngle (decimal) | The maximum angle limit in degrees for preserving the geometry without removing the corner points

Outputs:

  • Out (Polylines)

Reverse Direction

Reverses the direction of the input polylines.

Inputs:

  • In (Polylines)

Outputs:

  • Out (Polylines)

Spiral

Creates a continuous spiral from a list of polylines. The polylines need to be sorted, with unified direction and with their seams aligned. The spiral is created through interpolation of neighbour polylines.

Inputs:

  • In (Polylines)
  • Divisions (integer) | Number of divisions per polyline to define the resolution of the spiral. Higher number of divisions increases the resolution.

Outputs:

  • Out (Polylines)

Tween Meshes

Returns an interpolated list of meshes between two input meshes. The meshes need to be similar and have the same number of triangles.

Inputs:

  • InputA (Mesh)
  • InputB (Mesh)
  • Count (integer) | Number of interpolated meshes

Outputs:

  • Out (Meshes)

Tween Planes

Returns an interpolated list of planes between two input planes.

Inputs:

  • InputA (Plane)
  • InputB (Plane)
  • Count (integer) | Number of interpolated planes

Outputs:

  • Out (Planes)

Tween Polylines
